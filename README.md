 # BANK APP

 Bank_app is a console application of a bank that allows users to enter `pin` and do other transactions including:

 - Check Balance

 - Send Money

 - Check Mini Statement

# Build app
- mkdir build
- cd build
- cmake ..
- make

# run executable
./bank_app/bank_app

# NB
- had to install sqlite3
`brew install sqlite3`
- had to install bcrypt
`brew install bcrypt`
